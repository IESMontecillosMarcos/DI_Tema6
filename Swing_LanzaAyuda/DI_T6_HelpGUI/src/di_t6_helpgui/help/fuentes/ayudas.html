Solucion de problemas

>> No puedo escribir/cambiar datos personales usuario tras insertar el DNI.

No es posible borrar o modificar los datos de un usuario.

> Mis datos personales salen vacios.

Es debido a que no esta registrado. Introduzca sus datos correctamente.

> Inserto un usuario, pero los datos siguen ahi y no se borran.

Tras introducir los datos, pulse el boton limpiar.
